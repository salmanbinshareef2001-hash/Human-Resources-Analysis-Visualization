# 📊 HR Analytics Dashboard (Power BI)

## 🔍 Project Overview

This project presents an interactive HR Analytics Dashboard built using Power BI. The goal is to analyze employee data and identify key factors affecting employee attrition, job satisfaction, and workforce distribution.

---

## 📁 Dataset Information

The dataset contains employee-level information such as:

* Employee ID
* Age
* Gender
* Department
* Education Field
* Job Role
* Attrition (Yes/No)

---

## 📊 Key Metrics

* **Total Employees:** 1470
* **Attrition Count:** 237
* **Attrition Rate:** 16.12%
* **Active Employees:** 1233
* **Average Age:** 37

---

## 📈 Dashboard Features

* Department-wise Attrition Analysis
* Employee Distribution by Age Group & Gender
* Job Satisfaction Ratings by Job Role
* Education Field-wise Attrition
* Attrition Rate by Different Age Groups

---

## 🛠 Tools & Technologies Used

* Power BI (Data Visualization & Dashboard)
* Excel (Data Cleaning & Preparation)
* DAX (Data Analysis Expressions)

## 💡 Key Insights

* Highest attrition is observed in the R&D department
* Majority of employees are in the 25–34 age group
* Job satisfaction plays a significant role in employee retention
* Certain education fields show higher attrition trends

## 🚀 How to Use This Project

1. Download the `.pbix` file from this repository
2. Open it using Power BI Desktop
3. Interact with filters and visuals to explore insights

---

## 🎯 Business Use Case

This dashboard helps HR teams to:

* Identify high attrition areas
* Improve employee retention strategies
* Make data-driven HR decisions
